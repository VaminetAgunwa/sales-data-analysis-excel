# 📊 Sales Data Analysis Project (Excel Dashboard)

## 🔍 Project Overview

This project focuses on cleaning, analyzing, and visualizing sales data to uncover key business insights. The goal is to transform raw, messy data into a structured format and present actionable findings using an interactive dashboard.

---

## 📁 Dataset Description

The dataset contains sales transaction records including:

* Order ID
* Customer Name
* Region
* Product
* Order Date
* Sales Amount

The raw data included inconsistencies such as missing values, duplicate records, invalid entries, and inconsistent date formats.

---

## 🧹 Data Cleaning Process

The following steps were performed using Microsoft Excel:

* Removed duplicate records based on Order ID
* Handled missing values (customer names and sales)
* Corrected inconsistent date formats
* Cleaned invalid numeric entries in the sales column
* Standardized text fields (region and product names)

---

## 📊 Data Analysis

Key business questions answered:

* Which product generates the highest revenue?
* Which region performs best in terms of sales?
* What are the sales trends over time?

Pivot tables were used to aggregate and summarize the data.

---

## 📈 Dashboard Features

An interactive Excel dashboard was created with:

* Total Sales KPI
* Total Orders KPI
* Average Sales KPI
* Sales by Product (Bar Chart)
* Sales by Region (Pie Chart)
* Sales Trend Over Time (Line Chart)
* Slicers for dynamic filtering (Region & Product)

---

## 💡 Key Insights

* The **Laptop** category generated the highest revenue
* The **West region** recorded the strongest sales performance
* Sales showed consistent growth with noticeable peaks mid-month

---

* Microsoft Excel (Data Cleaning, Pivot Tables, Dashboard Creation)
